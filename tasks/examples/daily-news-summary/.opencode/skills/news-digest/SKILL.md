## What I do
- 搜索和聚合 AI/科技新闻来源
- 按重要性排序和筛选新闻
- 生成简洁的中文摘要（每条 2-3 句话）
- 输出格式化的 markdown 文件，包含日期、标题和来源

## When to use me
当任务需要收集和整理新闻资讯时使用此 skill。适用于日报、周报等定期新闻汇总场景。

## Workflow
1. 首先使用 web search 工具搜索今日 AI/科技新闻
2. 筛选出最重要的 5-10 条
3. 为每条撰写简洁的中文摘要
4. 按重要性排序
5. 输出为 markdown 格式文件，保存到 output/ 目录

## Output format
```markdown
# AI/科技新闻日报 — YYYY-MM-DD
生成时间：HH:MM

## 1. 新闻标题
2-3句话的核心要点概括。

**来源**: [来源名称]

---
```
